Kurt Gödel is My Dad – A 3D Experience

Explora un mundo tridimensional donde los cubos giran, cambian de color, y el cielo es el límite (bueno, casi).

Acerca del Proyecto

Este proyecto muestra cómo crear una aplicación 3D interactiva utilizando Three.js, integrada con un repository de GitHub para facilitar el acceso y la colaboración.

Usa el mouse para rotar el cubo, ajusta las configuraciones en el panel a la derecha, o haz clic en "Reiniciar" para restaurar la escena.
